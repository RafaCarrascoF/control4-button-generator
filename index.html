Herramienta de drivers

Generador de Control4Scenario experience button

Genera drivers .c4z personalizados a partir del driver base y una imagen PNG. Default/OFF se crea en blanco y negro; Selected/ON se mantiene en color.

Guía de uso

Genera drivers .c4z de scenario experience buttons para Control4 a partir de imágenes. Todo ocurre en tu navegador: no se sube nada a ningún servidor.

  1. Rellena Nombre del driver y Creador en cada botón.
  2. Elige el modo de imagen:
    • Una imagen: subes una imagen en color. El ON queda en color y el OFF se genera en blanco y negro automáticamente.
    • Dos imágenes: subes una imagen para el OFF (p. ej. puerta cerrada) y otra para el ON (p. ej. puerta abierta), ambas en color.
  3. Usa + Añadir botón si quieres generar varios a la vez.
  4. Pulsa Generar. Con un botón se descarga un .c4z; con varios, un .zip con todos.

Recomendado un PNG cuadrado en color. El ID del driver y el creador para el XML se calculan solos a partir de lo que escribas.
